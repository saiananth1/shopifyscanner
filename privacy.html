Privacy Policy for Shopify Store Scanner & Exporter

Last updated: [insert date]

Shopify Store Scanner & Exporter is a Chrome extension designed to help users inspect publicly available Shopify storefront information from websites they choose to scan.

Information accessed

The extension may access publicly available storefront information, including:

  • Product titles
  • Product handles
  • Product URLs
  • Collection titles
  • Collection handles
  • Collection URLs
  • Product vendors
  • Product types
  • Product tags
  • Public product and collection timestamps

Information not collected

The extension does not access or collect:

  • Passwords
  • Payment information
  • Checkout information
  • Customer account information
  • Private Shopify admin data
  • Cookies
  • Personal communications
  • Email content
  • Location data
  • Health information
  • Financial information

How information is used

The extension uses publicly available storefront information only to:

  • Detect whether the current website is a Shopify storefront
  • Count public products and collections
  • Display products and collections in searchable tables
  • Export products and collections to CSV
  • Show scan status and activity logs
  • Save the latest scan locally so the same store does not need to be rescanned every time the popup is opened

Local storage

The extension stores scan results locally in the user’s browser using Chrome extension storage and IndexedDB. This local storage is used only to improve usability and avoid repeated scans of the same store.

Data sharing

The extension does not sell, rent, transfer, or share user data with third parties.

External servers

The extension does not send scan results to an external server controlled by the developer. Requests are made directly from the user’s browser to the public website selected by the user.

User control

Users can clear stored scan data by clearing Chrome extension storage, using browser settings, or removing the extension.
